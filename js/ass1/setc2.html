Student Name and Image Interaction



Student Name Field:

If the name is entered, the font color will change to red and font size will increase to 18px.



Instructions:

  • Enter a student name to change the font color and size.
  • If the text box is empty and loses focus (onblur), the image will appear.
  • Hover over the image to enlarge it, click to make it bigger, and release the mouse button to return to its medium size.
